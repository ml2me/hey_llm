# hey_llm
### Pet проект "Эй, БЯМ"

###### Главная идея:
	Модульный ассистент, который будет достаточно простым, чтобы можно было его развернуть у себя и не бояться утечек.
	Модульность предполагает, что его части(навыки и этапы развёртывания) можно реализовывать последовательно.
	Для проекта в рамках курсов от ods.ai планиркуется использовать один из модулей  

###### Этапы:
	- Разворачивание сервера для персонального LLM.
	- Edge-приложение для активного слушания. Android приложение для выявления команды пробуждения далее аудио начинает стримится на свой сервер для стандартной работы в качестве LLM. Пока предполагается что активационная фраза будет "Эй, БЯМ" (это и дало название проекту).
	- Разворачивание мультиагентного ассистента.

###### Навыки: 
	- Учитель с памятью (контекстом), инициативностью (шедулером), персонифицированными объяснениями
	- Парсер-суммаризатор статей по теме
	- Озвучивание книги
